# emma.illustre — Portfolio artistique

Portfolio React/Vite construit autour du concept **« Entre le trait et la matière »**.

## Démarrer

```bash
npm install
npm run dev
```

## Œuvres

Les œuvres source de `Dessins / TRAVAUX EMMA` ont été converties en WebP (maximum 1920 px, qualité 82) et sont structurées par discipline dans `public/artworks/`.

| Dossier source | Dossier web | Œuvres intégrées |
| --- | --- | ---: |
| DESSIN | `drawing` | 7 |
| CROQUIS | `sketches` | 14 |
| ILLUSTRATIONS | `illustrations` | 7 |
| PHOTO | `photography` | 13 |
| PROJET SCOLAIRE | `school-projects` | 7 |
| DESIGN GRAPHIQUE | `graphic-design` | 6 |
| CROQUIS POUSSE | `traditional-sketches` | 10 |

Les noms d’origine sont conservés dans la convention de noms des fichiers et dans `src/portfolio.tsx`; les titres publics des fichiers horodatés sont volontairement neutres afin de ne pas inventer d’intention ou de contexte.

## À configurer avant mise en ligne

- ajouter une adresse e-mail de contact si nécessaire.
- renseigner les informations disponibles (année, technique, dimensions) pour les œuvres ;
- connecter le formulaire à Netlify Forms ou au service de messagerie choisi.
